About This Project

A Deep Learning Approach to Image Forgery Detection

Overview

This project implements a Convolutional Neural Network (CNN) for detecting image forgeries, inspired by the research paper "A Deep Learning Approach to Detection of Splicing and Copy-Move Forgeries in Images" by Y. Rao et al.

Image forgery detection is a critical task in digital forensics. With the advancement of image editing tools, it's becoming increasingly difficult to distinguish between authentic and manipulated images. This project aims to provide an automated solution using deep learning.

Methodology

SRM Filters

High-pass filters extract noise residual features from images, detecting manipulation inconsistencies.

CNN Architecture

Custom CNN extracts hierarchical features from image patches, capturing forgery patterns.

Feature Fusion

Multiple patch features are fused using max/mean pooling for comprehensive representation.

SVM Classification

SVM performs final binary classification to determine authenticity.

Datasets

CASIA2

A comprehensive dataset containing authentic and tampered images with various forgery types.

Accuracy: 96.82%

NC2016

NIST dataset for media forensics challenge with diverse manipulated images.

Accuracy: 84.89%
